# Evidence

This directory contains artifacts, scripts, and logs proving our evidence-first approach.

## Structure
- `run_chaos.py`: The simulation script to generate synthetic fraud cases and inject failures.
- `chaos_runs.json`: The output data from the chaos simulation.
- `plots/`: Directory for any generated visualization of the case metrics.
- `logs/`: Execution logs from agents and simulators.
- `coding-agent-sessions/`: Transcripts or logs of AI coding assistants used to help build the project.
